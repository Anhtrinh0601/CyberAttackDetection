# **CYBER ATTACK DETECTION**
This project is a part of Cybersecurity and Information Management Project at Dublin Business School (DBS). The project's goal is to propose a solution to cyber attacks aiming at Lloyd Banking Group in particular and financial institution in general.
The system includes 2 key modules which are cyberattack detection and traffic monitoring. While the first module is related to a machine learning model development in backend, the later involves dashboards and visualizations displayed in the interface.

## **_Datesets_**
Dataset for visualization: https://www.kaggle.com/datasets/teamincribo/cyber-security-attacks \
Dateset for model development: https://www.kaggle.com/datasets/hassan06/nslkdd
